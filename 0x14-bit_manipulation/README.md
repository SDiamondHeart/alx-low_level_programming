BITWISE OPERATORS AND BIT MANIPULATION

The bitwise operators in C programming are:

1. & - AND OPERATOR. It is a binary operator and its output is 1 whern both bits are 1, otherwise it is 0.

2. | - OR OPERATOR. It is also a binary operator, and its output is 0, when both bits are 0, otherwise it is 1.

3. ~ - NOT OPERATOR. It is a unary operator. This means that its output is 0 when the bit is 1 and vice versa.

4. << - LEFTSHIFT OPERATOR

5. >> - RIGHTSHIFT OPERATOR.
